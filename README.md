# TASK-02

------------------ Bitwise Operator Tasks (1–8) -----------------------

1. The program compares numbers 10 and 6 in binary form and prints the result where both numbers have 1 in the same position.

2. This compares 12 and 5 and prints result where any one number has 1 in binary.

3. This changes all bits of number 8 into opposite (like positive becomes negative style). So result becomes –9.

4. This compares 15 and 9 and prints result where bits are different.

5. This moves binary digits of 7 two places to the left, so the value becomes bigger (like multiplying by 4).

6. This moves binary digits of 20 one place to the right, so the value becomes smaller (like dividing by 2).

7. User enters two numbers and program finds AND result of those numbers.

8. User enters two numbers and program finds XOR result (difference in bits).

----------------------- String Tasks (9–14) ----------------------------

9. The word “hi” is printed 4 times continuously.

10. The word “python” is printed 3 times.

11. Two words “super” and “man” are joined together to form “superman”.

12. Three strings are joined to print “hello world”.

---------------------- Input and Type Casting Tasks (15–20) -----------------------

15. User enters name → program shows its data type, which is always string.

16. User enters age → program converts it into integer number and prints it.

18. User enters two marks → program finds average of marks.

19. User enters values of a and b → program calculates the formula result.

20. User enters a number → first program shows it as string type, then converts to integer type and shows again.

------------------- Unit Digit Tasks (21–25) ------------------------

21. User enters a number → program prints the last digit using string method.

22. Program finds last digit using modulus (%) operator.

-------------------- If Statement Tasks (26–30) -----------------------

26. Since 10 is greater than 5 → program prints message.

27. If user number is more than 50 → message is printed.

28. If age is 18 or above → person is eligible.

------------------- If-Else Tasks (31–34) ----------------------

31. Program checks whether number is even or odd.

32. If marks are 35 or more → Pass, otherwise Fail.

-------------------- Nested If Tasks (35–37) -------------------

35. Job Selection:
Program checks age first, then height, then weight.
Only if all conditions are correct → person is selected.

36. College Admission:
Program checks marks first, then age.
If both correct → admission granted.

37. Sports Selection:
Program checks age → height → weight.
If all are satisfied → selected.

Nested means one condition inside another condition.

------------------- Match Statement Tasks (38–40) ----------------------

39. User enters number from 1 to 3 → program prints color name.

40. User enters number from 1 to 4 → program prints fruit name.

Match works like switch case (choosing option).
